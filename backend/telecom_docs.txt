==== CONCEPT: 5G Technology ====
5G (Fifth Generation) is the latest wireless network standard providing peak data speeds up to 20 Gbps theoretical, with real-world speeds of 1-4 Gbps on mmWave and 100-500 Mbps on Sub-6GHz bands. Key advantages over 4G LTE: ultra-low latency under 1 millisecond (vs 20-50ms for 4G), massive device density supporting 1 million devices per square kilometer, and network slicing for dedicated virtual network segments. 5G uses three spectrum bands: Low-band (sub-1GHz, wide coverage), Mid-band (1-6GHz, balance of speed and coverage), and High-band mmWave (24GHz+, extreme speed, short range). 5G enables autonomous vehicles, remote surgery, AR/VR streaming, and smart city infrastructure.

==== CONCEPT: NB-IoT (Narrowband IoT) ====
NB-IoT is a 3GPP Low Power Wide Area Network (LPWAN) standard defined in Release 13. Operating in licensed spectrum (700MHz, 800MHz, 900MHz bands), it provides: extended battery life (10+ years on AA batteries), deep indoor penetration (20dB better than GPRS), low cost modules (under $5), and support for 50,000+ devices per cell. NB-IoT transmits in a narrow 200kHz channel with peak data rate of 250 Kbps downlink. Ideal for smart meters, environmental sensors, asset tracking, and smart agriculture where infrequent small data transmissions are needed.

==== CONCEPT: LTE-M (LTE for Machines) ====
LTE-M (also called Cat-M1) is a 3GPP LPWAN standard for IoT. Unlike NB-IoT, LTE-M supports voice calls (VoLTE), mobility/handover between cells, and higher data rates (1 Mbps). Uses 1.4MHz bandwidth with PSM (Power Saving Mode) and eDRX (Extended Discontinuous Reception) for battery optimization. Battery life up to 10 years. Supports FOTA (Firmware Over The Air updates). Best for mobile IoT assets: vehicle tracking, wearables, medical devices that need occasional higher throughput or mobility.

==== CONCEPT: VoLTE (Voice over LTE) ====
VoLTE enables high-definition voice calls transmitted as data packets over 4G/5G networks rather than using legacy 2G/3G circuit-switched voice. Benefits: HD voice quality (wideband audio at 16kHz vs 8kHz for 3G), faster call connection (under 1 second vs 5-8 seconds for 3G), simultaneous voice and high-speed data, 40% better spectral efficiency. VoLTE uses IMS (IP Multimedia Subsystem) core. When VoLTE is unavailable, phones fall back to CSFB (Circuit Switched Fallback) to 3G/2G for calls.

==== CONCEPT: eSIM ====
eSIM (Embedded SIM) is a programmable SIM soldered directly onto a device's motherboard. Unlike physical SIM cards, eSIM can be remotely provisioned with a carrier profile via RSP (Remote SIM Provisioning) using SM-DP+ servers. Supports multiple carrier profiles stored simultaneously (only one active). Benefits: no physical SIM swap needed, device activation anywhere globally, smaller form factor for wearables and IoT, carrier switching via software. eSIM follows GSMA SGP.02 (M2M) and SGP.22 (Consumer) specifications.

==== CONCEPT: Network Slicing ====
Network slicing is a 5G core network capability that creates multiple virtual networks (slices) on shared physical infrastructure. Each slice has dedicated guaranteed QoS parameters: bandwidth, latency, reliability. Example slices: eMBB (enhanced Mobile Broadband) for streaming, URLLC (Ultra-Reliable Low Latency Communications) for autonomous vehicles, mMTC (massive Machine Type Communications) for IoT. Slices are isolated — a congested consumer slice doesn't affect an enterprise URLLC slice. Managed via SDN (Software Defined Networking) and NFV (Network Function Virtualization).

==== CONCEPT: Static IP in Telecom ====
A static IP address is a fixed, permanent public IP address assigned to a SIM card that never changes across sessions. Unlike dynamic IPs (assigned fresh each session from a DHCP pool), static IPs enable: inbound connections to devices (hosting servers, remote desktop), consistent whitelisting in firewalls, reliable VPN endpoint configuration, and device management at known addresses. Critical for enterprise use cases: remote CCTV monitoring, ATM networks, industrial SCADA systems, branch office connectivity. Static IPs cost more due to IPv4 scarcity and dedicated routing table entries.

==== CONCEPT: Roaming ====
Mobile roaming enables a subscriber to use their home network subscription while physically present in another country or region. Two types: National Roaming (using another domestic carrier's network in coverage gaps) and International Roaming (using partner networks abroad). Roaming works via SS7/DIAMETER protocol messages between home network (HLR/HSS) and visited network (VLR). Wholesale roaming agreements between operators define inter-carrier settlement rates (IOT rates). Data roaming uses GPRS/LTE roaming via GRX/IPX interconnects. With 5G, roaming uses N17 interface and SEPP (Security Edge Protection Proxy).

==== CONCEPT: Data Rollover ====
Data rollover is a plan feature that carries unused data allocation forward to the next billing cycle rather than expiring it. Implementations vary: Full rollover (all unused data carried forward), Capped rollover (maximum rollover limit, e.g. 10GB cap), Expiry-based rollover (rolled data expires after 1 billing cycle). Rollover incentivizes plan upgrades and reduces subscriber churn. Telecom operators manage rollover accounting via PCRF (Policy and Charging Rules Function) and OCS (Online Charging System) nodes in the core network.

==== CONCEPT: QoS (Quality of Service) ====
QoS in telecommunications refers to traffic management mechanisms that prioritize network resources for specific applications or subscribers. Key QoS parameters: bandwidth (throughput guarantee), latency (delay bound), jitter (delay variation), and packet loss rate. In LTE/5G, QoS is managed via QCI/5QI (QoS Class Identifier / 5G QoS Indicator) values: QCI-1 for VoLTE (conversational voice), QCI-4 for gaming, QCI-9 for best-effort internet. The PCRF/PCF (Policy Charging and Rules Function) enforces QoS policies. Premium plans receive higher QCI values ensuring better experience during network congestion.

==== CONCEPT: MVNO (Mobile Virtual Network Operator) ====
An MVNO is a wireless service provider that does not own licensed spectrum or network infrastructure. Instead, MVNOs lease wholesale capacity from MNOs (Mobile Network Operators) and resell it under their own brand with value-added services. MVNO types: Full MVNO (owns core network, HLR/HSS, billing), Light MVNO (leases most network functions), Enhanced Service Provider (reseller with branded experience). MVNOs target niche markets: budget MVNOs (low cost), ethnic MVNOs (international calling), enterprise MVNOs (custom SLAs), IoT MVNOs (M2M focus). Examples: Google Fi (MVNO on T-Mobile/US Cellular), Boost Mobile (MVNO on AT&T).
